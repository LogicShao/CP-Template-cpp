# Templates

这个仓库总结了部分算法竞赛模板  
但困于个人能力限制，本仓库只有提高组级别的模板  
而且个别模板质量并不高，也希望各位多多包涵  
如果你有更好的模板或者想法，我很乐意合并你的请求  

另外，本仓库中大部分模板来自 `AcWing算法课` 和 `算法进阶指南`  
少部分来自网络或个人代码，感谢Ta们对我编程学习的帮助
